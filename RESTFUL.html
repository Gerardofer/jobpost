RESTFUL Express Routing

We will be using a recently created app to track jobs as an example for the routes.

Name Path HTTP Verb Purpose
Index /jobs GET List of all jobs
New /jobs/new GET Shows new jobs form
Create /jobs POST Creates a new job, then redirects someplace
Show /jobs/:id GET Shows information about one specific job
Edit /jobs/:id/edit GET Shows edit form from one job
Update /jobs/:id PUT Update a particular job, then redirects
Delete /jobs/:id Destroy Delete a particular job, then redirects
